# Project Brief

## Project Name
VIRT (go_to_virt)

## Core Purpose
A dating/matching platform that operates as a Telegram WebApp. Users browse profiles, like/dislike, get matched, and chat in real-time.

## Core Requirements
- User profile management with photos (AWS S3)
- Feed-based profile browsing with matching algorithm
- Like/dislike system with mutual match detection
- Real-time chat via WebSocket (Socket.io)
- Telegram bot integration for notifications and onboarding
- Blocking and complaint/moderation system
- Invitation/referral links
- Multi-language support (i18n)

## Clients
1. **Web frontend** — Angular 16 Telegram WebApp (primary client)
2. **Mobile app** — React Native (Android + iOS)
3. **Telegram bot** — Telegraf-based bot service

## Target Users
Russian-speaking audience using Telegram as the entry point.
